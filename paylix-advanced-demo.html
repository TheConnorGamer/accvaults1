Enhanced Features

This page demonstrates all available Paylix custom fields including gateway selection, blockchain options, email prefill, coupon codes, quantity control, variants, and custom styling.

Gateway Selection

Pre-select the payment gateway for your customers. Available options:

BITCOIN
ETHEREUM
LITECOIN
USDT
USDC
USDC_NATIVE
TRON
BINANCE_COIN
SOLANA
POLYGON
MONERO
BITCOIN_CASH
STRIPE
PAYPAL
SKRILL
PERFECT_MONEY
SUMUP
PADDLE
data-paylixecommerce-gateway="USDT"

Blockchain Selection

For cryptocurrencies with multiple blockchains, specify the preferred network:

BEP20
ERC20
MATIC
TRC20
data-paylixecommerce-blockchain="ERC20"

Checkout Step

Control which step customers see when opening the embed. Options: GATEWAY or CHECKOUT

  • GATEWAY - Shows product details with addons and coupon options
  • CHECKOUT - Skips directly to payment
data-paylixecommerce-step="CHECKOUT"

Email Prefill

Pre-fill the customer's email address for faster checkout:

data-paylixecommerce-email="customer@example.com"

Coupon Code

Apply a coupon code automatically. If invalid, no error will appear:

data-paylixecommerce-coupon="10_PERCENT_OFF"

Default Quantity

Set the default quantity for the product:

data-paylixecommerce-quantity="3"

Product Variant

Pre-select a specific product variant:

data-paylixecommerce-variant="Premium Edition"

Custom Styling

Apply custom CSS to match your brand. Two methods available:

  • HTML Head - Add CSS link with id="paylixecommerce-css"
  • Button Attribute - Use data-paylixecommerce-css
data-paylixecommerce-css="https://cdn.paylix.gg/static/css/custom-embed-styles.css"

Custom Fields

Hide or prefill custom fields from your Paylix product:

data-paylixecommerce-custom-discord="@username#1234"

Complete Example with All Fields

Here's a button with all custom fields configured:

<button
data-paylixecommerce-product="6908cfd78d4f3"
data-paylixecommerce-gateway="USDT"
data-paylixecommerce-blockchain="ERC20"
data-paylixecommerce-step="CHECKOUT"
data-paylixecommerce-email="customer@example.com"
data-paylixecommerce-coupon="VAULTS5"
data-paylixecommerce-quantity="1"
data-paylixecommerce-variant="Premium"
data-paylixecommerce-custom-discord="@user#0000"
data-paylixecommerce-css="https://cdn.paylix.gg/static/css/custom-embed-styles.css"
type="submit">
Purchase
</button>
