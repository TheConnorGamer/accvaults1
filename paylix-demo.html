Product Button

Use the Product Button for single product purchases. Replace PRODUCT_UNIQID with the unique ID of your product from the Paylix Dashboard. 

<button 
    data-paylixecommerce-product="PRODUCT_UNIQID"
    type="submit"
    alt="Buy Now with paylix.gg">
    Purchase
</button>

Integration Tips

  • The Paylix embed script is already loaded on this page
  • Buttons automatically trigger the Paylix payment modal
  • You can add custom classes and attributes to style the buttons
  • Both product and group buttons work with the same embed script
  • The alt attribute is optional but recommended for accessibility

Configuration

To set up Paylix integration on your site:

1. Include the Paylix embed script in your HTML:
<script src="https://cdn.paylix.gg/static/js/embed.js"></script>

2. (Optional) Include the embed CSS for better styling:
<link href="https://cdn.paylix.gg/static/css/embed.css" rel="stylesheet">

3. Add your button with the appropriate data attribute:
<button data-paylixecommerce-product="YOUR_PRODUCT_ID" type="submit">
    Buy Now
</button>

Need help? Check out the full integration guide or visit the Paylix documentation.
